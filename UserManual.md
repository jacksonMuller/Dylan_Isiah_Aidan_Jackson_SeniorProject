# ARM Control Manual

**Project:** Vision-Guided Robotic Arm  
**Version:** 1.0  
**Authors:** Aidan Hodges, Dylan Suzuki, Isaiah Pajarillo, Jackson Muller



## Instructions for the arm

### 1: First You Need to get your 3d printer files and servos from this github --> https://github.com/TheRobotStudio/SO-ARM100  (Guessing your not going to do this BJ)

### 2: Assemble your arm

### 3: Connect your arm to a raspberry pi and mother control board

### 4: Download Python to your raspberry pi

### 5: Once Connected, run keyboard_motor_control.py in the raspberry pi's terminal

### 6: Refer to the controls found here ![ARM Control Manual](Images/ARM%20Control%20Manual.png)
